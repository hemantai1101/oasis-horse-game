## Notes

- ⚠️ Val loss (0.707) is **higher** than depth-3 runs (~0.608) — **not a regression**
- Depth=4 labels encode deeper strategic value; harder to approximate from raw position features
- Model converged cleanly with no overfitting
- Root cause: **model too small** (14k params) for depth-4 function complexity
- Cannot compare val loss directly to Run 003 — different data distribution, different difficulty
- Next lever: widen architecture → 41→256→128→1
